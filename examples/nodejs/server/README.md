# Node.js Server Example

Server-side example demonstrating the bysquare library in Node.js with TypeScript.

## Quick Start

```bash
npm install
npm start
```

## Example Code

See [main.ts](main.ts) for the complete example.

## Runtime Compatibility

Works with multiple JavaScript runtimes:

**Node.js:**

```bash
npm start
```

**Bun:**

```bash
bun run main.ts
```

**Deno:**

```bash
deno run --allow-read main.ts
```
